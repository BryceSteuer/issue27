Program of the week

Switch by Python

Switch is a very fun and quite addicting game created by prestigious Python (see Interviews tab). Even though the graphics are not the game's best side, the gameplay, transitions, and overall clean feel of the game make up for this. The point of this Python-coded platformer is simple - get as far as possible by switching walls to avoid oncoming obstacles that may disrupt your progress. There are many things to look for as well as look out for, including "pups" which will move you up levels, as well as coins which increase your score. Overall, the pros outweigh the cons in this classic Python game. This game was rated by The Programming Times:
Complexity: 6.75/10
Fun: 9.25/10
Graphics: 4.5/10
Effects/Transitions: 7.75/10


Other reviews

Escape The Prison by Varun

Escape The Prison is a platformer game by Valun. In this game, as the title portrays, you must escape the prison: however, this is not easy and leads to hours of gaming through the twenty-five levels. Each level brings new treacherous dangers, including acid and lava, making it more difficult to reach the end portal and clear the level. The graphics are amazing for a platformer, and the transitions are pretty well done. The only downside is the lag, which may be too much for older browsers. This game was rated by The Programming Times:
Complexity: 4.25/10
Fun: 7.75/10
Graphics: 6.50/10
Effects/Transitions: 7.75/10


3D Pong by Sy

3D Pong is a game from Sy. It portrays the classic ping-pong table ready to play in three dimensions. The graphics are not very realistic, although the 3D effect is impressive. AS for the gameplay, you do compete with the CPU, which can sometimes be tough - and very laggy.
This game was rated by The Programming Times:
Complexity: 7.75/10
Fun: 6.5/10
Graphics: 2.75/10
Effects/Transitions: 6.75/10

About The Programming Times

The Purpose

The purpose of The Programming Times is to provide the Khan Academy programming community with reliable news and keep users up-to-date with the latest programs and events. At the time of founding, The Programming Times was the only newspaper doing this. Now that there are many other newspapers, this need is not as pertinent.

The History

The Programming Times was founded on January 4, 2018 by Bryce S. The first issue was released the following weekend. It racked in only five viewers, a start of obscurity. Now, The Programming Times is Khan Academy's longest running newspaper. No other Khan Academy newspaper has gotten to 25 issues. Over the months, The Programming Times has achieved quite a prestigious name as a deliverer of reliable news.

A "one man show"

The Programming Times was founded and continues to be ran by one person and only one - Bryce S. Please be lenient if you do not get an issue when promised, as running a weekly newspaper and doing the website design for it is a bit overwhelming. I do not accept sponsors; however, if you would like to post the latest issue on your subscription page, feel free.

Contact

You can contact The Programming Times on Khan Academy weekdays from 3pm to 5pm Central Standard Time on Khan Academy.
